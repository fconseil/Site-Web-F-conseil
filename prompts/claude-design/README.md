# Claude Design importé

Ce dossier contient le contenu extrait du fichier `.claude/Importer le prompt.zip`.

## Fichiers importés
- `Site Franck.dc.html` : projet Claude Design principal, contenant le template et la logique de rendu.
- `F-CONSEIL-site.html` : version HTML du site.
- `support.js` : script JavaScript nécessaire pour le projet Claude Design.
- `assets/` : ressources image du site.
- `uploads/` : image de logo supplémentaire.

## Objectif
Ce dossier permet de conserver le prompt et les fichiers associés directement dans le workspace pour pouvoir consulter, réutiliser ou modifier le projet.

## Que faire maintenant
- Ouvre `prompts/claude-design/Site Franck.dc.html` pour voir le contenu du prompt Claude Design.
- Ouvre `prompts/claude-design/support.js` si tu veux inspecter la logique utilisée par le projet.
- Si tu veux, je peux aussi extraire le prompt sous forme de fichier texte `prompts/claude-design/claude-design-prompt.txt`.

## Remarque
Ce dossier est une copie de l’import fourni, il n’est pas encore intégré à une commande `.claude` existante. Si tu veux, je peux ajouter une commande de gestion dans le workspace pour accéder plus rapidement à ce prompt.
