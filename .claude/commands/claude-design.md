# /claude-design

> Commande pour accéder au prompt Claude Design importé et obtenir un résumé de son contenu.

## Mission

Quand je lance `/claude-design`, exécute la séquence suivante :

1. Informe-moi que le prompt Claude Design a été importé dans `prompts/claude-design`.
2. Fais un résumé court du contenu clé du fichier `prompts/claude-design/claude-design-prompt.txt`.
3. Indique les fichiers principaux disponibles :
   - `prompts/claude-design/Site Franck.dc.html`
   - `prompts/claude-design/claude-design-prompt.txt`
   - `prompts/claude-design/support.js`
   - `prompts/claude-design/README.md`
4. Propose une action suivante claire, par exemple :
   - ouvrir `prompts/claude-design/claude-design-prompt.txt`
   - extraire un passage précis
   - convertir le prompt en un format plus lisible

## Règles importantes

- Répond en français.
- Sois direct et synthétique.
- Ne modifie aucun fichier sans validation.
